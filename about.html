About the Project

This project is a Fashion E-commerce Website built as part of my NTI training program. The goal of the project is to provide users with a simple, modern, and responsive online shopping experience.

The website allows users to browse products, view details, and manage a shopping cart. To ensure a secure shopping experience, products can only be added to the cart after logging in. The cart also uses LocalStorage to save items even if the browser is closed or refreshed.

Key Features

Add to Cart

Users can add products from the store directly to their cart.

Persistent Cart

Shopping cart items are saved using LocalStorage even after closing the browser.

Login Required

Only logged-in users are allowed to add products to the cart.

Image Slider

Interactive Swiper.js slider to showcase products in a stylish way.

Responsive Design

The website is fully responsive and works smoothly on desktop and mobile devices.

SweetAlert2

Used SweetAlert2 library to show modern and stylish alerts instead of default browser alerts, for better user experience in actions like login and adding products to cart.
